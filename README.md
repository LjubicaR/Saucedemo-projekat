# SauceDemo Projekat
SauceDemo is an e-commerce demo website designed for practicing automated tests.  
The main goal of this project is to simulate adding products to the cart and completing the shopping process.

## Dependencies

Operating System: Windows 10
- IDE: IntelliJ IDEA Community Edition 2023.2.2
- Supported Browsers: Chrome (mandatory), Firefox, Edge, Safari (optional)

## Setup Instructions

Open the terminal in your IDE and clone the repository

git clone https://github.com/LjubicaR/Saucedemo-projekat/tree/main

## Included Tests

1. **LoginTest** – validates the login functionality
2. **AddToCartTest** – adds a product to the shopping cart
3. **RemoveItemFromCartTest** – removes a product from the cart
4. **ProductDetailsTest** – opens the product details page and verifies information
5. **CheckoutTest** – performs the checkout process and validates the overview
